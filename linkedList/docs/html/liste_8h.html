void addNodeToTail ( L_Ptr * l,
char c )

Aggiunge un elemento in coda alla lista.

Questa funzione aggiunge un nodo con un carattere specificato alla fine della lista. Se la lista è vuota, crea il primo nodo.

Parametri
lPuntatore alla testa della lista. Il puntatore sarà aggiornato se necessario per mantenere la lista collegata correttamente.
cIl carattere da aggiungere come nuovo nodo alla lista.

◆ printLista()

void printLista ( L_Ptr l)

Stampa ogni elemento di una lista dalla testa alla coda.

Questa funzione stampa ogni carattere contenuto nei nodi della lista a partire dal primo nodo (head) fino all'ultimo (tail).

Parametri
lPuntatore alla testa della lista. Se la lista è vuota, non stampa nulla.

void reverseList ( L_Ptr * l)

Inverte l'ordine della lista.

Questa funzione inverte l'ordine degli elementi della lista es: a->b->c => c->b->a

Parametri
lpuntatore alla testa della lista. Al termine della funzione l punterà alla coda, che sarà la nuova testa.
